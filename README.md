This project is a peronal sandbox to experiment with ARM developing.
Mainly the application will be bare metal and Linux.
The emulation will be done mainly with QEMU.

## qemu-asm-test 

For Debian/Ubuntu, install the needed packages. Issue the following command:

sudo apt install qemu-system-aarch64 gdb-multiarch binutils build-essential

Install the Aarch64 bare-metal target (aarch64-none-elf) toolchain for your 
host machine. Binaries may be downloaded on the following link:

https://developer.arm.com/downloads/-/arm-gnu-toolchain-downloads
